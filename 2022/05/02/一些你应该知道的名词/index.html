一些你应该知道的名词

发表于2022-05-02|更新于2022-05-02|实用知识(杂谈)
|字数总计:539|阅读时长:2分钟

UML：UML（Unified Modeling Language的缩写）统一建模语言,是用来对软件密集系统进行可视化建模的一种语言。

URL：统一资源定位符（URL，英语UniformResourceLocator的缩写）也被称为网页地址，是因特网上标准的资源的地址。

URI：Web上可用的每种资源-HTML文档、图像、视频片段、程序等-由一个通用资源标志符（Universal ResourceIdentifier,简称"URI"）进行定位。安卓也是。

XML：XML（ExtensibleMarkup Language）即可扩展标记语言，它与HTML一样，都是SGML(StandardGeneralized Markup Language,标准通用标记语言)。

YAML：YAML是"YAMLAin’t Markup Language(YAML不是一种标记语言)"的缩写。

C/S结构：即Client/Server(客户机/服务器)结构，是大家熟知的软件系统体系结构，通过将任务合理分配到Client端和Server端，降低了系统的通讯开销，需要安装客户端才可进行管理操作。

B/S结构：B/S（Browser/Server结构）结构即浏览器和服务器结构。

Aspect-Oriented Programming：面向切面编程（AOP）

just-in-time compilation：即时编译，JIT 编译

NFC：近距离无线通信（Near Field Communication）

SVG：Scalable Vector Graphic，可伸缩矢量图（SVG）

SDK：软件开发工具箱（Software Development Kits）

NDK：Native Development Kit，是 Android的一个工具开发包。作用：快速开发C、 C++的动态库，并自动将so和应用一起打包成 APK

JNI：Java Native Interface，即 Java本地接口

JNI与NDK

SSL：Secure Socket Layer，安全套接层（SSL）

SSO：单点登录（Single Sign-On）

TCP：传输控制协议（Transmission Control Protocol）

User Datagram Protocol，用户数据报协议（UDP）

User Interface：用户界面，包括视觉设计、交互设计和用户体验（UI）

VIP：虚拟 IP（或不译）

VPN：虚拟专用网络（Virtual Private Network）

Inversion of Control：控制反转（IoC）

Internet Protocol：互联网协议（IP）

Hyper Text Transfer Protocol：超文本传输协议（HTTP）

Hyper Text Transfer Protocol Secure：超文本传输安全协议（HTTPS）

File Transfer Protocol：文件传输协议（FTP）

Content Delivery Network：内容分发网络（CDN）

Command-Line Interface：命令行界面（CLI）

Garbage Collector：垃圾回收（GC）

Secure Shell： 安全外壳协议（远程连接作用）（SSH）
